Nursery-Backend API Docs

Base URL: http://localhost:3000 (your port may differ based on PORT)
Auth: Protected routes accept JWT token either via:
1) Header: Authorization: Bearer <token>
2) Cookie: token=<token>
Error response: { "message": "..." }

Auth APIs

Route What it does Example
POST
http://localhost:3000/auth/register
 Creates user and returns token + sets cookie token. 
{
  "name": "Rajat Singh",
  "email": "rajat@example.com",
  "password": "your_password_here"
}
POST
http://localhost:3000/auth/login
 Logs in user and returns token + sets cookie token. 
{
  "email": "rajat@example.com",
  "password": "your_password_here"
}
GET
http://localhost:3000/auth/me
 Returns current user (requires auth token). 
Authorization: Bearer <token>
POST
http://localhost:3000/auth/logout
 Clears auth cookie and logs out. 
Authorization: Bearer <token>

Products APIs

Route What it does Example
GET
http://localhost:3000/products
 Query params: search, categoryId, active, minPrice, maxPrice, page, limit 
http://localhost:3000/products?search=&categoryId=&active=&minPrice=&maxPrice=&page=1&limit=20
GET
http://localhost:3000/products/:id
 Get a product by Mongo ObjectId. 
http://localhost:3000/products/<productId>
POST
http://localhost:3000/products/add
 Admin-only. Creates product. 
Authorization: Bearer <adminToken>
{
  "name": "Product name",
  "price": 100,
  "discount": 10,
  "soldBy": "Company/Dealer",
  "responseRate": 80,
  "categoryId": "60f...ObjectId",
  "description": "Short description",
  "image": ["https://example.com/img1.jpg", "https://example.com/img2.jpg"],
  "stock": 25,
  "isActive": true
}
PUT
http://localhost:3000/products/:id
 Admin-only. Updates only fields you send. 
http://localhost:3000/products/<productId>
Authorization: Bearer <adminToken>
{
  "price": 120,
  "description": "Updated description",
  "stock": 18
}

Sales APIs

Route What it does Example
POST
http://localhost:3000/sales
 Admin-only. Creates sale and reduces product stock. 
Authorization: Bearer <adminToken>
{
  "productId": "60f...ObjectId",
  "quantity": 3
}
GET
http://localhost:3000/sales
 Query params: page, limit, from, to, productId, adminId 
http://localhost:3000/sales?page=1&limit=20&from=&to=&productId=&adminId=
Authorization: Bearer <adminToken>
GET
http://localhost:3000/sales/:id
 Get sale by id (ObjectId). 
http://localhost:3000/sales/<saleId>
Authorization: Bearer <adminToken>
